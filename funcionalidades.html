    Campos de Proteção de Dados

    Os campos de Proteção de Dados servem para coletar autorizações dos Leads (titulares dos dados) para o tratamento dos dados pessoais que acontece parcialmente ou totalmente por meio do RD Station Marketing.

    Relação com as leis

    LGPD e GDPR determinam que, para uma empresa realizar qualquer operação com um dado pessoal, é necessário ter uma base legal, ou seja, uma autorização da lei. Por isso, essa funcionalidade permite que as estratégias de aquisição de Leads sejam realizadas dentro das melhores práticas da lei.

    Saiba como usar a funcionalidade

    Gerenciamento de Cookies

    Para que você possa rastrear o comportamento dos visitantes do seu site ou Landing Page, é preciso uma notificação ou autorização do titular de dados para uso de cookies. Com os dois tipos de banners disponíveis, você avisa seus visitantes sobre uso e Política de cookies e também controla o uso desses dados de acordo com o consentimento obtido.

    Relação com as leis

    As leis de privacidade de dados consideram Cookies como um dado pessoal. Dessa forma, assim como você precisa atribuir uma Base Legal a um Lead, você também precisa coletar a autorização dos visitantes do seu site ou Landing Page para monitorar o comportamento deles.

    Saiba como usar a funcionalidade

    Direitos do Titular

    Desativar Lead

    Caso um Lead entre em contato com a sua empresa pedindo para interromper o tratamento dos dados pessoais para fins de comunicações de marketing, é possível desativar este contato. O Lead também pode se descadastrar nos emails disparados pelo RD Station Marketing.

    Relação com as leis

    As legislações de Proteção de Dados garantem ao Lead o direito de solicitar a interrupção do processamento dos seus dados pessoais em alguns casos. Um exemplo é a oposição ao tratamento de dados quando a Base Legal utilizada não for a de Consentimento.

    Saiba como usar a funcionalidade

    Editar dados do Lead

    Os dados pessoais do contato podem ser atualizados de várias formas dentro do RD Station Marketing. A cada nova conversão, atualizando por importação ou mesmo manualmente.

    Relação com as leis

    As legislações de Proteção de Dados garantem aos Leads o direito de solicitar a correção de dados pessoais quando estiverem incompletos, inexatos ou desatualizados. A correção de ser imediata, considerando a LGPD e em até um mês, segundo a GDPR. Este direito é conhecido como Direito de Retificação ou Direito de Edição.

    Saiba como usar a funcionalidade

    Exclusão de Leads

    Caso um Lead peça para interromper o Processamento dos Dados Pessoais, você pode excluir o contato.

    Relação com as leis

    O titular de dados tem o direito de solicitar a exclusão dos dados pessoais que sua empresa tem dele. No entanto, o direito de exclusão não é aboluto e depende do contexto do pedido. Por isso, nem sempre se aplica.

    Saiba como usar a funcionalidade

    Exportação de dados de Leads

    Exporte dados de um Lead específico (ou de vários) com as informações de bases legais referentes a cada um deles.

    Bases Legais Alterações em Massa

    Caso você tenha uma lista de contatos no CRM que não estão com a Base Legal devidamente registrada. Você pode editar e adicionar essa informação para vários contatos ao mesmo tempo a partir dos filtros. Isso facilita a atualização das informações caso algo tenha sido importado errado anteriormente

    Saiba como usar a funcionalidade

    Bases Legais Criação Manual

    Ao cadastrar manualmente um novo contato, você já pode adicionar as informações de Base Legal. Evitando que um novo contato fique sem essa informação.

    Permite que você configure que todos os contatos gerados a partir de uma integração via API já tragam a informação de Bases Legais preenchidas. Sem precisar preenche-lás uma a uma quando entrarem na sua conta do RD Sation CRM
